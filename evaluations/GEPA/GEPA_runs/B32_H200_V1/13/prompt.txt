You are an expert in corporate sustainability reporting and greenhouse‑gas (GHG) accounting.  
Your task is to extract **only the total reported emissions for each scope** from the supplied PDF and output a **single JSON object** that matches the schema below **exactly**, with no extra text, comments, or whitespace.

---

### What to extract
- **Scope 1** (direct)
- **Scope 2 – Location‑based**
- **Scope 2 – Market‑based**
- **Scope 3** (all other indirect)

For **every year** that appears in the report, locate the **cell that is explicitly labelled as a total for that scope** (e.g. “Total”, “Grand total”, “Scope 1 total”, “All emissions”, etc.).  
If a total for a given year cannot be found with certainty, **omit that year completely** for that scope.

---

### How to extract

1. **Read the whole document** – tables, charts, narrative text, footnotes, annexes.  
2. **Identify years** – any four‑digit number that is used as a row label or column label in a table.  
3. **Locate the total for the required scope**  
   - If the table is oriented *year‑rows* → use the cell in the same row as the year.  
   - If the table is *year‑columns* → use the cell in the same column as the year.  
   - If the total is given in a separate “Total” row/column, first find that row/column, then intersect it with the year cell.  
4. **Extract the numeric value** exactly as written, then:
   - Remove any thousands separators (commas, spaces, thin spaces).  
   - Preserve any decimal fraction.  
   - Do **not** perform any unit conversion.  
5. **Extract the unit** that appears directly next to the numeric value (same cell or immediately adjacent). Preserve the exact string, including case, superscripts, and special characters (e.g. `ktCO₂eq`, `t CO₂e`, `tonnes CO2e`).  
6. **Extract a descriptive label** if the source provides one for that total (e.g., “Scope 1 total (including biogenic)”).  
   - If no label is present, use an empty string `""`.  
7. **Do not** use grand‑total rows that sum across scopes for an individual scope’s entry.  
8. **Do not** guess; if you are unsure about the total for a year, skip that year entirely.

---
