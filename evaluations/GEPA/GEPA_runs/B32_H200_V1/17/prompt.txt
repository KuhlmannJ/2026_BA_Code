### Definition of “total”
A total is an entry that is explicitly labelled as the overall amount for the scope for a given year. Acceptable labels (case‑insensitive, may contain extra text) include, but are not limited to:
- “Total”, “Grand total”, “Scope 1 total”, “Scope 2 total”, “Scope 2 (location‑based) – Total”, “Scope 2 (market‑based) – Total”, “Scope 3 total”, “All emissions”, “Grand total emissions”, “Total emissions”.

If a table contains both a breakdown and a grand‑total for the same year, **use only the grand‑total**. If no grand‑total can be identified with certainty, **omit that year completely** – never guess or output a partial value.

### Extraction procedure
1. **Search the entire document** – tables, figures, footnotes, and narrative text.
2. **Determine table orientation**:
   - If years appear in the leftmost column (or first column after row headers) → treat years as **row labels**.
   - If years appear in the top row (or first row after column headers) → treat years as **column labels**.
3. **Locate the total cell for the required scope**:
   - *Row‑label orientation*: find the cell in the same row whose column header (or the first column of that row) contains a total‑type label.
   - *Column‑label orientation*: find the cell in the same column whose row header contains a total‑type label.
4. **Extract the numeric value exactly as printed**, then:
   - Remove any thousands separators (commas, spaces, thin spaces).
   - Preserve any decimal fraction; output as a JSON numeric literal (e.g., `2071.0`).
5. **Extract the unit string** that appears immediately adjacent to the number
   (e.g., `ktCO₂eq`, `tonnes CO2e`, `t CO2e`, `tCO₂e`). Keep the string unchanged; do not convert units.
6. **Extract the label**: capture the exact wording that identifies the total
   (e.g., “Scope 1 total (including biogenic)”). If no extra label exists, set
   `"label"` to an empty string `""`.

- Shifting values to the wrong year (e.g., using 2020’s value for 2021). Ensure the year‑value pairing matches the table orientation described above.
