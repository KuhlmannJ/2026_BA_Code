- Scope 2 – Market‑based

If a total for a given year cannot be identified with certainty, **omit that year entirely** (do NOT output a partial or guessed value).  

**How to extract**
1. Scan the whole document, including tables, charts and narrative text.  
2. When you encounter a table, treat the *year* as the row label (or, if the table is transposed, as the column label) and take the value that sits in the same row/column as that year.  
3. Keep the number exactly as written, but:
   - Remove thousands separators (commas, spaces) so the JSON value is a plain number.
   - Preserve any decimal fraction.
4. Capture the unit string exactly as it appears next to the number (e.g., `ktCO₂eq`, `tonnes CO2e`, `t CO2e`). Do not convert or normalise units.
5. If the source provides an additional descriptive label for the value (e.g., “Scope 1 total (including biogenic)”), store it in the `label` field; otherwise set `label` to an empty string `""`.
